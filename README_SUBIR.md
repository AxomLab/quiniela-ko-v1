# Quiniela KO Web Optimizada v2

Subir a la raíz del repo `AxomLab/quiniela-ko-v1` y reemplazar:

- `index.html`
- `script.js`
- `styles.css`

No reemplazar `data/quiniela.json`; lo debe seguir actualizando n8n.

## Correcciones v2

- El corredor ahora usa `--progress` con zona segura de inicio y meta.
- Los avatares ya no deben salirse por la izquierda en celular.
- Se incorpora la carga robusta de avatares basada en la versión estable de fase de grupos: cache-busting, pre-carga y reintentos.
- Se mantienen los textos y estructura KO.
